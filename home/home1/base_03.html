Модули

Модули предлагают способ расширить программирование на Python. Вы можете создать свои модули, использовать те, которые поставляются с Python или которые облегчают написание программ. Список модулей, поставляемых с Python, находится по адресу:
http://docs.python.org/modindex.html. Некоторые модули предназначены для определённых операционных систем, но большинство из них кроссплатформенные и работают одинаково на Linux , Mac и Microsoft Windows . Для использования внешнего модуля необходимо импортировать его в программу. Один из модулей, поставляемый Python, называется "random". Он даёт возможность генерировать псевдослучайные числа. Мы используем этот модуль в нашем первом примере.


#=======================================
# random_ex.py
# Module example using the random module
#=======================================
import random
# print 14 random integers
for cntr in range (1,15):
print random.randint (1,10

Давайте разбурём код по строкам. Первые четыре строки - это комментарии, о которых мы говорили в прошлый раз. Пятая строка говорит Python, что нужно использовать модуль "random". Мы должны явно указать Python на это.

Фенкции

Когда мы импортировали модуль random, мы использовали функцию randint(). Функция - это блок кода, который создан для того, чтобы вызывать его в программе, обычно, более одного раза. Это упрощает работу над программой и предотвращает набор одного и того же кода снова и снова. Грубо говоря, если вам нужно написать некоторый код больше, чем 1 - 2 раза, стоит сделать его функций. Хотя следующие два примера очень простые, это хорошие примеры функций. Теперь мы хотим, скажем, взять два числа, сложить их, перемножить, вычесть одно их другого, затем вывести на экран эти числа и результат одновременно. Допустим, на нужно сделать это три раза для трёх наюоров чисел. Тогда наш пример будет выглядеть как код ниже.

#silly example

print '%d * %d = %d ' % (1,2,1*2)

print '\n'
